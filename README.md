#### How to Style List Markers

```html
<ul>
  <li><strong>Total</strong>: Approximately 10 minutes</li>
  <li><strong>Preparation</strong>: 5 minutes</li>
  <li><strong>Cooking</strong>: 5 minutes</li>
</ul>
```
```css
ul li::marker {
    font-size: 0.8rem;
    color: var(--Brown-800);
}
```
